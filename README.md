# CLI
CLI for The Programming Language

## Installation
**POSIX**
```sh
curl -fsSL https://cdn.thelang.io/cli | sh
```

**Windows Batch file**
```batch
powershell -exec bypass -c "(iwr 'https://cdn.thelang.io/cli-win').Content | cmd /Q"
```

**Windows PowerShell**
```powershell
(iwr 'https://cdn.thelang.io/cli-win').Content | cmd /Q
```

## Usage
```sh
$ the -h
```
